This is a visualiser for the mathematical puzzle of the Knight's Tour. The knight must visit each square of the chessboard exactly once to complete the tour. My algorithm uses Warnsdorff's heuristic alongside the additions made by mathematician Douglas Squirrel to complete the tour for all the board sizes available here (and more). If you want to try the tour yourself, see this link: https://www.maths-resources.com/knights/.
